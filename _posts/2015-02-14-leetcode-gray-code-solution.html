The gray code is a binary numeral system where two successive values differ in only one bit.
Given a non-negative integer n representing the total number of bits in the code, print the sequence of gray code. A gray code sequence must begin with 0.
For example, given n = 2, return [0,1,3,2]. Its gray code sequence is: 
00 - 0
01 - 1
11 - 3
10 - 2
Note:
For a given n, a gray code sequence is not uniquely defined.
For example, [0,2,3,1] is also a valid gray code sequence according to the above definition.
For now, the judge is able to judge based on one instance of gray code sequence. Sorry about that. Analysis:
If we write down the gray code of 3 bits, we will find the pattern. 
000
001
011
010
110
111
101
100
Did you see that? The last 4 numbers are just the reverse of first 4 numbers add 2^2=4. The first 4 numbers are 0, 1, 3, 2. The reverse of them are 2, 3, 1, 0. So the last 4 numbers will be 2+4, 3+4, 1+4, 0+4, say 6, 7, 5, 4. Alright, once we know the pattern, it's easy to write code.

Code:
